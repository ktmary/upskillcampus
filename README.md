# Prediction of Agriculture Crop Production in India

## Internship Details

- **Internship Domain:** Data Science and Machine Learning
- **Organization:** upSkill Campus
- **Project Title:** Prediction of Agriculture Crop Production in India

## Project Overview

This project uses Machine Learning techniques to analyze agricultural crop data and predict crop yield based on various features such as crop type, cultivation cost, production cost, state, and yield information.

The project includes data preprocessing, exploratory data analysis, visualization, model training, and prediction using the Random Forest Regression algorithm.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Dataset

The dataset contains agricultural crop information collected from Government of India open data sources. It includes crop production, cultivation cost, yield, and state-wise agricultural information.

## Machine Learning Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Engineering
5. Model Training
6. Model Evaluation
7. Prediction
